# AntFlow Web Video Editor Guide

The Web Video Editor is a fully-featured non-linear editor (NLE) running in your browser.

## 1. Interface Overview

### The Layout
-   **Media Library (Left)**: Access Uploads, Stock Video, Audio, Text, and Transitions.
-   **Preview Player (Center)**: Watch your edit in real-time.
-   **Properties Panel (Right)**: Adjust settings for the selected clip (Position, Scale, Opacity, Effects).
-   **Timeline (Bottom)**: Multi-track sequencer for arranging your content.

## 2. Working with Media

### Importing
1.  Go to the **Media** tab on the left sidebar.
2.  Click **"Upload"** to add files from your computer.
3.  Alternatively, drag and drop files directly into the browser window.

### The Timeline
-   **Adding Clips**: Drag a file from liquid Media Library onto a Track in the timeline.
-   **Trimming**: Hover over the edge of a clip and drag to shorten or extend it.
-   **Splitting**: Position the playhead (red line) and click the **Split** (Scissors) icon to cut a clip.
-   **Moving**: Drag clips left/right or between tracks to reorder.
-   **Zooming**: Use the slider or `Ctrl + Scroll` to zoom in/out of the timeline.

## 3. Editing Features

### Text & Titles
1.  Click the **Text** tab.
2.  Drag a text style (Basic, Animated, Subtitle) to the timeline (usually on a track *above* your video).
3.  Double-click the preview text or use the Right Panel to edit the content, font, and color.

### Transitions
1.  Click the **Transitions** tab.
2.  Drag a transition (e.g., Fade, Dissolve, Slide) *between* two adjacent clips on the same track.
3.  Adjust the transition duration by dragging its edges.

### Effects & Color
1.  Select a clip in the timeline.
2.  In the **Right Panel**, scroll to **Effects**.
3.  Enable **Color Correction** to adjust Brightness, Contrast, and Saturation.
4.  Apply filters like **Blur**, **Glitch**, or **Chroma Key** (Green Screen).

## 4. Keyframing (Advanced)
*Animate properties over time.*
1.  Select a clip.
2.  In the Right Panel, look for the **Diamond Icon** next to a property (e.g., "Scale").
3.  Move the playhead to the start of the action and click the diamond to set a **Keyframe**.
4.  Move the playhead forward and change the value (e.g., Scale to 150%). Another keyframe is auto-added.
5.  Play back to see the smooth animation.

## 5. Exporting

1.  Click **"Export"** in the top right corner.
2.  **Settings**:
    -   **Resolution**: 720p, 1080p, 4K.
    -   **Format**: MP4 (H.264).
    -   **Quality**: High, Medium, Low.
3.  Click **"Start Export"**.
4.  Once rendering is complete, the file will automatically download. You can also share it directly to social platforms.
