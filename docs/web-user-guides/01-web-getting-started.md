# Getting Started with AntFlow Web

Welcome to the AntFlow Web Studio! This comprehensive platform allows you to create, edit, and stream professional content directly from your browser.

## 1. Accessing the Studio

1.  **Navigate**: Open your web browser (Chrome or Edge recommended) and go to `https://antflow.io` (or your deployed URL).
2.  **Login/Sign Up**: 
    -   Click the "Login" button in the top right.
    -   Use your Email or Single Sign-On (Google/GitHub/Apple) to access your account.

## 2. The Dashboard

The Dashboard is your command center.

### Sidebar Navigation
-   **Dashboard**: Overview of recent activity and quick actions.
-   **Projects**: Your video editing projects library.
-   **Live Studio**: Access the streaming control center.
-   **Voice Lab**: Clone voices and generate TTS.
-   **Assets**: Manage uploaded media (Videos, Images, Audio).
-   **Team**: Manage members and permissions (if you are an admin).
-   **Settings**: Configure profile, billing, and API keys.

### Quick Actions
Top cards allow you to instantly:
-   **New Project**: Start a video editor session.
-   **Go Live**: Launch the streaming studio.
-   **Record**: Capture screen or camera usage.

## 3. Managing Projects

### Creating a Project
1.  Click **"New Project"**.
2.  Select **Aspect Ratio**: 16:9 (Landscape), 9:16 (Portrait), or 1:1 (Square).
3.  Choose a **Template** (optional) or start blank.
4.  Enter a **Project Name**.

### Project Library
-   **Folders**: Organize projects into folders for better management.
-   **Search**: Find projects by name or date.
-   **Actions**: Click the "..." menu on any project to Rename, Duplicate, or Delete.

## 4. Account Settings

Click your avatar in the top right to access:
-   **Profile**: Change your display name and password.
-   **Billing**: View subscription plans and usage limits.
-   **API Keys**: Generate keys for developer integration.

## Support
Need help? Click the **"?"** icon in the bottom right to access the Help Center or contact support.
